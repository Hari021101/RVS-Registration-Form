## Quick Start

### For Team Members Filling the Form

1. **Open the Application**

   - The form will open in your web browser
   - You'll see the title "RVS Team Member Registration"

2. **Navigate the Form**

   - The form has 7 steps shown at the top
   - Fill out the current section
   - Click "Next" to move to the next section
   - You can click on any completed step (with ‚úì) to go back and edit

3. **Understanding Field Types**
   - Fields with a red asterisk (\*) are **REQUIRED**
   - Fields with an "Optional" badge can be left empty
   - All inputs have helpful placeholder text showing the expected format

## Section-by-Section Guide

### Step 1: General Information üë§

**All fields are required**

- **Name**: Enter your full name
- **Gender**: Select from dropdown
- **Father Name / Mother Name**: Enter parents' names
- **Date of Birth**: Select from calendar picker
  - ‚ö° **Pro Tip**: Age will be calculated automatically!
- **Age**: Auto-filled, but you can edit if needed
- **Blood Group**: Select from dropdown (A+, B+, O+, AB+, etc.)
- **Marital Status**: Select from dropdown

### Step 2: Contact Details üìß

**All required except Current Address**

- **Personal Email**: Enter valid email (e.g., name@example.com)
- **Phone Number**: Enter 10-digit mobile number
- **Emergency Contact**: Enter 10-digit emergency contact number
- **Current Address**: (Optional) Your current residential address
- **Permanent Address**: Your permanent address

### Step 3: Government ID Details üÜî

**Aadhar and PAN are required**

- **Aadhar Number**: Enter 12-digit Aadhar number
  - Format: 123456789012
- **PAN Number**: Enter 10-character PAN
  - Format: ABCDE1234F (5 letters, 4 numbers, 1 letter)
- **Passport Number**: (Optional) If you have a passport
- **Driving License**: (Optional) If you have a license

### Step 4: Bank Details üè¶

**All fields are required**

- **Bank Name**: Name of your bank (e.g., State Bank of India)
- **Account Holder Name**: Name as per bank records
- **Account Number**: Your bank account number
- **IFSC Code**: 11-character bank IFSC code
  - Format: SBIN0001234
- **Branch Name**: Your bank branch name

### Step 5: Educational Qualification üéì

#### 10th Standard (Optional)

- Qualification, School Name, Board
- Year of Passing, Percentage/CGPA

#### 12th/Diploma (Optional)

- Qualification, School/College Name, Board
- Year of Passing, Percentage/CGPA

#### Under Graduate - **REQUIRED**

- College Name
- Degree (e.g., B.E., B.Tech, B.Sc)
- Department (e.g., Computer Science)
- Year of Passing
- Percentage/CGPA

#### Post Graduate (Optional)

- If not applicable, you can enter "N/A" or leave blank
- College Name, Degree, Department
- Year of Passing, Percentage/CGPA

### Step 6: Skills & Technical Details üíª

**Required Fields:**

- **Technology Expertise**: Your technical domains (e.g., Web Development, AI/ML)
- **Programming Languages**: Languages you know (e.g., Python, JavaScript)
- **Tools**: Development tools you use (e.g., Git, Docker)
- **Soft Skills**: Communication, Teamwork, etc.
- **Project Experience**: Brief description of projects

**Optional Fields:**

- **Soft Technical Skills**: Agile, Scrum, etc.
- **Certificates**: Professional certifications
- **Internship Certificates**: Previous internship details

### Step 7: Additional Information ‚ÑπÔ∏è

**Required:**

- **Known Languages**: Languages you speak (e.g., English, Hindi, Tamil)
- **LinkedIn URL**: Your LinkedIn profile link

**Optional:**

- **Hobbies**: Your interests and hobbies
- **GitHub URL**: Your GitHub profile (if you have one)

## After Submission

### Success Screen üéâ

After clicking "Submit" on the last step, you'll see:

1. **Confirmation Message**: "Registration Successful!"

3. **Download Options**:
   - Click **"Download Excel"** to save data as an Excel spreadsheet (.xlsx)

## Common Questions

### ‚ùì What if I make a mistake?

- You can navigate back to any previous step by clicking on it in the progress bar
- Edit the information and continue

### ‚ùì Can I save my progress?

- Currently, data is not saved automatically
- Complete the form in one session
- After submission, download the Excel or JSON file for your records

### ‚ùì What happens to my data?

- Your data is only stored in your browser's memory
- No data is sent to any server
- Download the Excel or JSON file to keep a personal copy

### ‚ùì Required vs Optional fields?

- Look for the red asterisk (\*) - these are REQUIRED
- "Optional" badge - you can skip these if not applicable
- The form won't let you proceed if required fields are empty

### ‚ùì I see a red border on a field?

- This means the field has an error
- Check the red text below the field for the specific issue
- Common issues:
  - Empty required field
  - Invalid format (email, phone, PAN, etc.)
  - Wrong number of digits

### ‚ùì Age is not calculating?

- Make sure you select a valid Date of Birth
- Age should calculate automatically
- If not, you can enter it manually

### ‚ùì The form won't let me proceed?

- Check for any fields with red borders
- Make sure all required fields are filled
- Check error messages below fields
